# 17. Implement Queue using Stacks

## 題目
用兩個 stack 實作 queue,支援 push、pop、peek、empty 操作。

## 面試重點
此題測試資料結構轉換的理解,特別是如何用 LIFO 模擬 FIFO。
面試官會觀察是否理解 amortized O(1) 的概念,以及何時進行元素轉移。

## 複雜度
- 時間複雜度:push O(1),pop/peek amortized O(1)
- 空間複雜度:O(n)

- 能否用一個 stack 實作 queue?
- 若用 queue 實作 stack,該如何設計?
