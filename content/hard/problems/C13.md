# 13. Longest Palindromic Substring

## 題目
給定字串 s,找出其中最長的回文子字串。

## 面試重點
此題測試**中心擴展與 Manacher 演算法**,展現從 O(n²) 到 O(n) 的優化思路。

**方法一:中心擴展 O(n²)**
- 遍歷每個可能的中心點
- 分奇數長度(單一中心)和偶數長度(雙中心)
- 從中心向兩邊擴展

**方法二:動態規劃 O(n²)**
- dp[i][j] = s[i:j+1] 是否為回文
- 轉移:dp[i][j] = (s[i] == s[j]) && dp[i+1][j-1]

**方法三:Manacher 演算法 O(n)**
- 預處理:在字元間插入特殊符號,統一奇偶
- 利用回文的對稱性,減少重複計算
- 維護當前最右回文邊界和中心

面試官會重點觀察:
- 中心擴展的實作:需要處理奇偶兩種情況
- DP 方法的填表順序:對角線方向
- 能否解釋 Manacher 的核心思想:利用對稱性
- Manacher 的實作難度較高,通常不強求
- 各方法的空間複雜度差異

**常考公司**:
- Google、Facebook/Meta
- Microsoft、Amazon
- 字串處理常考
- 通常要求 O(n²) 即可,Manacher 為加分項

此題展現了**從暴力到優化的完整思路**,是字串問題的經典。

## 複雜度
**中心擴展**:
- 時間複雜度:O(n²)
- 空間複雜度:O(1)

**動態規劃**:
- 時間複雜度:O(n²)
- 空間複雜度:O(n²)

**Manacher**:
- 時間複雜度:O(n)
- 空間複雜度:O(n)
