# 5. Add Two Numbers

## 題目
兩個非空鏈結串列代表兩個非負整數,數字以逆序儲存,每個節點包含一位數字。
將兩數相加並以相同形式回傳。

## 面試重點
此題測試**鏈結串列的同步遍歷與進位處理**,核心難點:
1. 處理不同長度的串列
2. 正確維護進位(carry)
3. 最後可能還有進位需要新增節點

面試官會重點觀察:
- 是否用 dummy node 簡化頭節點處理
- 迴圈條件:l1 or l2 or carry,確保所有情況都處理
- 取值時的空指標檢查:val1 = l1.val if l1 else 0
- 最後進位的處理:迴圈結束後 carry 可能為 1
- 指標移動的時機:確保不會 NullPointerException

此題展現了**鏈結串列操作的三大技巧**:dummy node、同步遍歷、邊界條件檢查。

## 複雜度
- 時間複雜度:O(max(m, n))
- 空間複雜度:O(max(m, n)),輸出串列的長度
