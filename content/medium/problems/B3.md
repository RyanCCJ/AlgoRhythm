# 3. Product of Array Except Self

## 題目
給定整數陣列 nums,回傳陣列 answer,其中 answer[i] 等於 nums 中除了 nums[i] 之外所有元素的乘積。
要求不使用除法,且時間複雜度為 O(n)。

## 面試重點
此題測試**前綴積與後綴積**的空間優化技巧,是理解「分解子問題」的典型案例。核心思路:
1. answer[i] = (i 左邊所有數的乘積) × (i 右邊所有數的乘積)
2. 用兩次遍歷分別計算左積和右積
3. 進階:用輸出陣列本身儲存左積,再用單一變數累積右積,達到 O(1) 額外空間

面試官會重點觀察:
- 是否理解為何不能用總乘積除以當前元素(存在 0 的情況,且題目禁用除法)
- 能否清楚解釋前綴積的概念:left[i] = nums[0] × ... × nums[i-1]
- 空間優化:如何將兩個輔助陣列優化為 O(1) 額外空間
- 對 0 的處理:雖然題目保證在 O(n) 時間內可解,但要思考 0 的影響

此題展現了如何通過**改變遍歷方向和累積變數**來優化空間複雜度。

## 複雜度
- 時間複雜度:O(n)
- 空間複雜度:O(1),不計入輸出陣列
