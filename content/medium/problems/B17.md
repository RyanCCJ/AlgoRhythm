# 17. Word Break

## 題目
給定字串 s 和字典 wordDict,判斷 s 是否可以被分割成字典中的單詞序列。
同一單詞可以重複使用。

## 面試重點
此題測試**字串 DP 與記憶化搜尋**,核心是子問題的定義與重疊性識別。

**方法一:動態規劃**
- 狀態定義:dp[i] = 字串 s[0:i] 是否可被分割
- 狀態轉移:dp[i] = true,若存在 j < i 使得 dp[j] = true 且 s[j:i] 在字典中
- 時間 O(n² × m),m 為平均單詞長度

**方法二:記憶化 DFS**
- 遞迴檢查從當前位置開始的子字串
- 用 memo 記錄已計算的結果
- 更直觀但可能堆疊溢位

面試官會重點觀察:
- 能否識別這是 DP 問題:有重疊子問題
- 狀態定義的清晰度:dp[i] 代表的是前綴而非後綴
- 雙層迴圈的意義:外層遍歷結束位置,內層尋找分割點
- 字典查找優化:用 Set 而非 List,O(1) 查找
- 剪枝:若 s[j:i] 長度超過字典最長單詞,可跳過
- 邊界:dp[0] = true 的含義(空字串可被分割)

此題展現了**如何將字串問題轉化為 DP**,特別是前綴型 DP。

## 複雜度
- 時間複雜度:O(n² × m),n 為字串長度,m 為平均單詞長度
- 空間複雜度:O(n)
